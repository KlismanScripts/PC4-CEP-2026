🎯 Objetivo del análisis

Evaluar la estabilidad del proceso de producción de remaches mediante gráficas de control X‑barra (media) y R (rango). Se determinará si el proceso está en control estadístico y si la variabilidad es aceptable.

📐 Parámetros del diseño

  • Tamaño de muestra: n = 4 remaches por muestra.
  • Número de muestras: k = 20.
  • Variable de interés: Longitud del remache (mm).
  • Límites de control: Basados en el rango promedio (R̄).
02
Identificación de la carta
Tipo de gráfica y factores

Se utiliza una carta de control X‑barra y R para variables continuas, apropiada para monitorear la media y la variabilidad de un proceso cuando se dispone de subgrupos de tamaño fijo (n = 4). Los límites de control se calculan a partir del rango promedio (R̄) y los factores constantes para n = 4:

  • A₂ = 0.729 (para límites de X‑barra)
  • D₃ = 0 (límite inferior para R)
  • D₄ = 2.282 (límite superior para R)
📌 Fórmulas empleadas:
  • X̄‑barra: LCS = X̄̄ + A₂·R̄  |  LCI = X̄̄ − A₂·R̄  |  Central = X̄̄
  • R: LCS = D₄·R̄  |  LCI = D₃·R̄  |  Central = R̄

05
Interpretación
Análisis de la estabilidad del proceso
📊 Gráfica X‑barra

La media de cada subgrupo oscila entre 1.390 y 1.4068, valores que se mantienen dentro de los límites de control [1.3820, 1.4145]. No se observan tendencias, ciclos ni puntos fuera de los límites. Esto indica que la media del proceso es estable y no ha sufrido desplazamientos significativos durante el periodo de muestreo.

📉 Gráfica R

Los rangos varían entre 0.004 y 0.042, todos por debajo del límite superior (0.0509) y por encima del límite inferior (0). No hay señales de aumento en la variabilidad, lo que sugiere que la dispersión del proceso es consistente y controlada.

✅ Cumplimiento de reglas

No se detectan violaciones de las reglas de Western Electric (puntos fuera de los límites, rachas de 7 puntos consecutivos por encima o debajo de la central, etc.). El proceso opera bajo causas comunes de variación.

06
Conclusiones
Recomendaciones finales
🔍 Estabilidad confirmada

Ambas cartas de control (X‑barra y R) indican que el proceso de producción de remaches se encuentra en control estadístico. No se requiere ninguna acción correctiva inmediata.

📏 Capacidad del proceso

La media general (1.3983 mm) y la variabilidad (R̄ = 0.0223) son consistentes. Se recomienda calcular los índices de capacidad (Cp, Cpk) para evaluar si el proceso cumple con las especificaciones técnicas.

🔄 Plan de acción

Mantener el esquema de muestreo actual (cada 10 minutos, n=4). Actualizar los límites de control periódicamente (cada 25-30 muestras) para reflejar posibles mejoras o cambios en el proceso.

🏭 Recomendación operativa

El proceso está bajo control y puede continuar su operación normal. Se sugiere mantener el plan de muestreo (cada 10 minutos) y actualizar periódicamente los límites de control a medida que se obtengan más datos. Si se desea mejorar la capacidad, se podría analizar la variabilidad entre máquinas o turnos.

📌 Nota: Este análisis asume que los datos son independientes y la distribución es aproximadamente normal. La gráfica R es adecuada para n = 4.
