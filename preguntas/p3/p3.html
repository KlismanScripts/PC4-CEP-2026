Problema 03

Análisis de capacidad – Brazo robótico (corto vs. largo plazo)

01
Descripción del problema
Contexto y datos del proceso

📌 Enunciado: Un brazo robótico clasifica paquetes según su desviación (en mm) respecto al centro del punto de colocación objetivo. La tolerancia de ingeniería es ±0.5 mm (LSL = -0.5, USL = 0.5, target = 0). Durante 5 días se tomaron subgrupos de n = 5 mediciones cada hora (25 subgrupos en total, dentro de cada turno) para evaluar la variación de corto plazo (within). Además, se calculó la variación total agregando los datos de los 5 días para evaluar el largo plazo (overall), pues se sospecha que hay diferencias entre turnos/días (cambios de calibración).

Estadístico Valor
X̄ (media de medias) 0.02 mm
R̄ (rango promedio, n=5, d₂=2.326) 0.18 mm
Desviación estándar global (s, N=125) 0.15 mm

🎯 Objetivo del análisis

Calcular y comparar los índices de capacidad de corto plazo (Cp, Cpk) y largo plazo (Pp, Ppk) para el brazo robótico de clasificación, e interpretar la diferencia en el contexto de operación 24/7 con distintos turnos.

📐 Parámetros del análisis

  • Variable: Desviación respecto al centro (mm).
  • Límites de especificación: LSL = -0.5, USL = 0.5, Target = 0.
  • Tamaño de subgrupo: n = 5, d₂ = 2.326.
  • Índices a calcular: Cp, Cpk, Pp, Ppk.
02
Identificación del análisis de capacidad
Corto plazo vs. Largo plazo

Se realiza un análisis de capacidad para datos normales con dos enfoques:

  • Corto plazo (within): Estima la variabilidad dentro de subgrupos usando σwithin = R̄/d₂. Evalúa la capacidad potencial del proceso cuando está bajo control.
  • Largo plazo (overall): Estima la variabilidad total usando la desviación estándar global s. Evalúa el desempeño real del proceso a lo largo del tiempo.
📌 Fórmulas empleadas:
  • σwithin = R̄ / d₂ = 0.18 / 2.326 = 0.0774
  • σoverall = s = 0.15
  • Cp = (USL − LSL) / (6·σwithin)
  • Cpk = min[(X̄ − LSL)/(3·σwithin), (USL − X̄)/(3·σwithin)]
  • Pp = (USL − LSL) / (6·s)
  • Ppk = min[(X̄ − LSL)/(3·s), (USL − X̄)/(3·s)]

04
Interpretación
Comparación de capacidad y análisis contextual
📊 Índices de corto plazo (Cp, Cpk)

Cp = 2.15 → La variación dentro de subgrupos (σwithin = 0.0774) es muy pequeña en relación con la amplitud de la especificación (1.0 mm). El proceso tiene una capacidad potencial excelente.
Cpk = 2.07 → El proceso está bien centrado (X̄ = 0.02, muy cerca de 0). La capacidad real dentro de turnos es excelente.

📉 Índices de largo plazo (Pp, Ppk)

Pp = 1.11 → La variabilidad total (s = 0.15) es mayor, lo que reduce la capacidad a un nivel moderado. El proceso es capaz pero no holgado.
Ppk = 1.07 → Aunque el proceso sigue centrado, la capacidad real a largo plazo es apenas superior a 1, lo que indica que hay variación adicional entre turnos que afecta el desempeño sostenido.

🔍 Comparación numérica y explicación

La diferencia es significativa: Cp (2.15) ≈ 1.94 × Pp (1.11) y Cpk (2.07) ≈ 1.93 × Ppk (1.07).
Esto ocurre porque el corto plazo mide la variación dentro de subgrupos homogéneos (mismo turno, mismas condiciones), mientras que el largo plazo incluye la variación entre subgrupos (cambios de turno, calibraciones, etc.). En un entorno 24/7 como Amazon, los cambios entre turnos siempre añaden variabilidad, por lo que el largo plazo es más realista.

05
Conclusiones
Recomendaciones y contexto operativo
✅ Proceso capaz (dentro de turnos)

El brazo robótico tiene una capacidad excelente cuando se controlan las condiciones del turno (Cp=2.15, Cpk=2.07). No hay problemas de centrado significativos.

⚠️ Desempeño real limitado por turnos

La capacidad real a largo plazo (Ppk=1.07) es significativamente menor que la de corto plazo. La variación entre turnos añade aproximadamente 0.0726 mm de desviación estándar adicional, lo que reduce la capacidad sostenida.

📊 ¿Cuál reportar a un cliente interno?

Para evaluar si el sistema de clasificación es confiable de forma sostenida en operación 24/7, se debe reportar Pp y Ppk (largo plazo). El corto plazo (Cp, Cpk) es útil para entender la capacidad del proceso cuando está bajo control, pero el largo plazo refleja el desempeño real que el cliente experimentará día a día, considerando los cambios entre turnos y calibraciones.

🏭 Recomendación operativa

Estandarizar los procedimientos de calibración entre turnos para reducir la variación entre subgrupos. Si se logra reducir la variación overall, Pp y Ppk se acercarán a Cp y Cpk. Mientras tanto, reportar Pp/Ppk para evaluaciones de confiabilidad sostenida.

📌 Nota: Cp/Cpk siempre será ≥ Pp/Ppk, porque σwithin ≤ σoverall. La magnitud de la diferencia refleja el impacto de las fuentes de variación entre subgrupos (turnos, lotes, calibraciones). En procesos 24/7, la capacidad de largo plazo es la métrica más relevante para el negocio.
