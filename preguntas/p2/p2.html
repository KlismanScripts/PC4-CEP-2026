Problema 02

Carta u – Errores de picking (tamaño de muestra variable, límites variables)

01
Descripción del problema
Contexto y datos del proceso

🎯 Objetivo del análisis

Monitorear la tasa de errores de picking (defectos por pedido) utilizando una carta de atributos tipo u con límites de control variables (por día), dado que el tamaño de muestra varía día a día.

📐 Parámetros del análisis

  • Tipo de carta: u (defectos por unidad).
  • Tamaño de muestra: Variable (ni).
  • Variable de interés: Número de errores de picking (defectos).
  • Límites de control: Variables, calculados para cada día usando su propio ni.
02
Identificación de la carta
Justificación de la elección (u vs p, np, c)

Dado que el tamaño de muestra varía día a día (ni no es constante) y el equipo audita un número variable de pedidos, se utiliza la carta u (defectos por unidad). La justificación es:

  • np requiere tamaño de muestra constante y cuenta unidades defectuosas (no defectos).
  • p se usa para proporción de unidades defectuosas, no para conteo de defectos múltiples.
  • c requiere área de oportunidad constante (mismo número de unidades inspeccionadas).
  • u es la carta apropiada para conteo de defectos por unidad cuando el área de oportunidad (en este caso, el número de pedidos auditados) varía. Los límites se calculan de forma variable para cada día, usando la fórmula con ni.
📌 Fórmulas para la carta u con límites variables:
  • Línea central: ū = (Σ errores) / (Σ ni)
  • LCSi: ū + 3 · √(ū / ni)
  • LCIi: max(0, ū − 3 · √(ū / ni))
  • ui: erroresi / ni

Nota: Los límites variables se ajustan al tamaño de muestra de cada día, lo que da mayor precisión cuando n fluctúa.

📌 Parámetros de la carta u
  • ū = 60 / 927 = 0.0647 (6.47 errores por cada 100 pedidos)
  • Límites variables: LCIi = max(0, ū − 3·√(ū/ni))
  • LCSi = ū + 3·√(ū/ni)
  • Cada día tiene sus propios límites según su ni.
📌 Ventaja de límites variables
  • Los límites se ajustan al tamaño de muestra de cada día.
  • Días con más pedidos tienen límites más estrechos (mayor precisión).
  • Evita falsas señales por variación en n.

04
Desarrollo en Minitab
Carta u con tamaños de muestra variables

En Minitab se utilizó la opción Stat → Control Charts → Attributes charts → U Chart. Se ingresaron los datos de la siguiente manera:

  • Columna con los defectos (errores) para cada día.
  • En el campo "Subgroup sizes" se especificó la columna que contiene los tamaños de muestra (ni), lo que permite que Minitab calcule límites variables para cada día.

Con esta configuración, Minitab genera límites de control que se ajustan al tamaño de muestra de cada subgrupo, reflejando la variabilidad real del proceso.

Interpretación
Estabilidad del proceso y análisis de resultados

Todos los puntos (tasas diarias) se encuentran dentro de los límites de control variables. No se observan puntos fuera de control ni patrones sistemáticos. El proceso de errores de picking se considera bajo control.

📈 Tasa media de errores

La tasa media estimada es ū = 0.0647 errores por pedido, lo que equivale a aproximadamente 6.5 errores por cada 100 pedidos. Este valor es un punto de referencia para la mejora.

✅ Cumplimiento de reglas

No se detectan violaciones de las reglas de Western Electric. El proceso opera bajo causas comunes de variación. Los límites variables han permitido un ajuste preciso a los diferentes tamaños de muestra.

06
Conclusiones
Recomendaciones finales
✅ Proceso en control estadístico

La carta u indica que el proceso de errores de picking se encuentra bajo control. No se requieren acciones correctivas inmediatas.

📊 Capacidad del proceso

La tasa media de errores es de 0.0647 por pedido. Si el objetivo de calidad es inferior a este valor, se deben implementar mejoras (ej. capacitación, mejoras en el sistema de picking) y luego monitorear la carta para verificar la reducción.

🔄 Plan de acción

Mantener el monitoreo diario con la carta u de límites variables. Si se detectan puntos fuera de control en el futuro, investigar causas (ej. cambios en el personal, sistema de picking, etc.). Se sugiere también calcular la capacidad del proceso (Pp, Ppk) para cuantificar el desempeño.

🏭 Recomendación operativa

El proceso está bajo control. Si se desea reducir la tasa de errores, se recomienda implementar un plan de mejora (ej. estandarización de procedimientos, auditorías adicionales) y monitorear el impacto mediante la carta u.

📌 Nota: La carta u con límites variables es la herramienta adecuada cuando el tamaño de muestra cambia. Esta aproximación proporciona límites precisos para cada día, evitando falsas alarmas por variaciones en n.
